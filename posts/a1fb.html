vue中main.js App.vue router.js等文件的简单介绍

2020-12-20
#Vue
Vue

一、 main.js

​ main.js是程序入口文件，在此文件中初始化实例，设定实例挂载的位置，并使用需要的插件及加载各种公共组件。

// 初始化实例并使用需要的插件,加载各种公共组件。
// 设定实例需要挂载的位置
import Vue from 'vue'
import App from './App.vue' // 根组件
import router from './router/router'
import './plugins/element.js'
import './assets/css/global.css' // 全局样式

// 导入字体图标
import './assets/fonts/iconfont.css'

import axios from 'axios' // 导入axios包
axios.defaults.baseURL = 'http://timemeetyou.com:8889/api/private/v1/' // 配置请求的根路径
Vue.prototype.$http = axios // 将axios包挂载到vue的原型对象上，然后每一个vue组件都可以通过this直接访问到$http (),从而去发起ajax请求

Vue.config.productionTip = false

new Vue(&#123;
  router, // 挂载路由
  render: h => h(App) // render函数的作用就是将App那个vue组件渲染成DOM节点,放到页面中去展示。
&#125;).$mount('#app')

/** 另一种写法
 * new Vue(&#123;
 *    el: '#app',
 *    router,
 *    render: h => h(App)
 * &#125;)
 */

​ import…from…指引用一些包和组件，from后面跟路径格式，若只给出vue， axios这样的包名，则会自动到node_modules中加载。

二、App.vue

​ App.vue是根组件，也是页面入口文件，所有的页面都在App.vue下进行切换。它负责构建定义及页面组件归集。

  </div>

​ 在App.vue中有一个标签，用来做路由的占位符，由路由匹配到的组件会渲染在里展示。

四、element.js

​ element.js中导入一些element组件

import Vue from 'vue'

// 导入element组件

Vue.use(FormItem)
Vue.use(Input)
