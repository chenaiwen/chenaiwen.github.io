#Vue
Vue

一. 简介

​ Vue 是一套构建用户界面的渐进式框架，它只关注视图层， 采用自底向上增量开发的设计，目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

学习Vue之前需要掌握：

​ html css javascript ajax

创建一个Vue

步骤：

  1. 导入Vue
<!-- 开发环境版本，包含了有帮助的命令行警告 -->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  1. 写html
<div id="app">
        &#123;&#123;message&#125;&#125;
</div>
<!--&#123;&#123;&#125;&#125;：插值表达式；模板表达式-->
```



3. 写js

 ```html
<script>
        let app= new Vue(&#123;
            el: '#app',
            data: &#123;
                message: 'Hello Vue!'
            &#125;
        &#125;)

</script>

 ```



##### data：数据对象

vue中用到的数据定义在data中。data中可以写复杂类型的数据。渲染复杂的数据类型时，遵守js语法即可，列如对象的点语法，数组的索引语法。



##### el:挂载点

通过css选择器设置vue实例管理元素，设置完后el命中的元素内部中两个大括号括起来的部分就会被data中同名的数据给替换掉。



##### 几个小问题：

1. Vue实例的作用范围？

   el命中的元素及元素内部的后代元素,外部不可以。

   

2. 是否可以使用其他选择器？

   可以，class也能用，直接选择标签也可以，但是只支持双标签，不支持单标签。 一般推荐使用id选择器，语义清晰。

   

3. 是否可以设置其他dom元素？

   可以使用双标签，不能使用html和body。

用Vue开发网页

Vue指令：

  1. 内容绑定，事件绑定

​ v-text v-html v-on

  1. 显示切换，属性绑定

​ v-show v-if v-bind

  1. 列表循环，表单元素绑定

v-text

  1. 设置标签内的文本值（textContent）。无论内容是什么，只会解析为文本。

  2. 默认写法会替换掉原来的值，使用差值表达式可以替换指定内容。

  3. 支持内部写表示式。

<script>
       var app= new Vue(&#123;
           //el:".app",
           el:"div",
           data:&#123;
              message:"Vue!",
              info:"前端"
           &#125;
       &#125;)
</script>

<div id="app" class="app">
        <h2 v-text="message">合肥</h2>  <!--覆盖掉标签内的值-->
        <h2 v-text="info">合肥</h2>     <!--覆盖掉标签内的值-->
        <h2>&#123;&#123;message+"!"&#125;&#125;合肥</h2>    <!--这种写法不会覆盖-->
</div>
字符串拼接：
<h2 v-text="message+'!'">合肥</h2>  
<h2 v-text="info+'!'">合肥</h2>     
<h2>&#123;&#123;message+"!"&#125;&#125;合肥</h2>

v-html

设置标签的innerHTML，如果内容中有html结构，html就可以被直接解析出来。如果只是文本，和v-text功能一样。

<script>
       var app= new Vue(&#123;
           //el:"#app",
           el:"div",
           data:&#123;
              message:"Vue",
              content:"<a href='#'>程序员</a>",
              info:"前端"
           &#125;
        &#125;)
</script>

<div id="app">
        <p v-text="content"></p>  <!--v-text只返回文本值-->
        <p v-html="content"></p>
</div>

v-on

为元素绑定事件,事件名不需要写on。指令可以简写为@。绑定的方法定义在methods属性中。方法内部通过this关键字可以访问定义在data中的数据。

<script>
       let app = new Vue(&#123;
           el:"#app",
           data:&#123;
               food:"西蓝花炒蛋"
           &#125;,
           methods:&#123;  //method是方法集合
               doIt:function()&#123;    //doIt是方法名
                   alert("做IT");
                &#125;,

               changeFood:function()&#123;
                   //console.log(this.food);
                   this.food+="好好吃!"  //this可以调用和修改data里的数据
                &#125;
           &#125;
       &#125;)
</script>

<div id="app">
     <input type="button" value="v-on指令" v-on:click="doIt">
     <input type="button" value="v-on简写" @click="doIt">
     <input type="button" value="双击事件" @dblclick="doIt">
     <h2 @click="changeFood">&#123;&#123;food&#125;&#125;</h2> <!--点击调用changeFood方法-->
</div>

v-show

根据表达式值的真假，切换元素的显示和隐藏。原理是修改元素的display，实现显示隐藏。

指令后面的内容，最终都会解析为布尔值。值为true元素显示，false元素隐藏。

数据改变之后，对应元素的显示状态会同步更新。

<div id="app">
    <img src="images/2031653.jpg" v-show="true">  <!--这种写法是不可以更改的-->
    <img src="images/325530.jpg" v-show="isShow"> <!--这种写法可以在data里面更改的-->
    <img src="images/325530.jpg" v-show="age>=18"> <!--根据表达式的值决定是否显示-->

    <!--点击自动切换-->
    <input type="button" value="切换显示状态" @click="changeIsShow"> 
</div>

<script>
    let app=new Vue(&#123;
      el:"#app",

      data:&#123;
        isShow:true,
        age:16
      &#125;,

      methods:&#123;
        changeIsShow:function()&#123;
            this.isShow=!this.isShow;
        &#125;  
      &#125;
    &#125;)
</script>

v-if

根据表达值的真假，切换元素的显示和隐藏（操纵dom元素）。

v-show操纵的是样式，v-if操纵的是dom。

当表达式的值为true，元素存于dom树中，false，从dom树中移除。

使用时，平凡切换时使用v-show，反之用v-if。

<div id="app">
    <p v-if="true">我是一个p标签</p>   <!--布尔值-->
    <input type="button" value="显示/隐藏" @click="toggleIsShow">
    <p v-if="isShow">我是一个p标签</p>  <!--函数-->
    <h2 v-if="temperature>=35">热死啦</h2>  <!--表达式-->
</div>




<script>
    let app=new Vue(&#123;
      el:"#app",

      data:&#123;
        isShow:false,
        temperature:36
      &#125;,

      methods:&#123;
        toggleIsShow:function()&#123;   //点击切换isShow的值
          this.isShow=!this.isShow;
        &#125;,        
      &#125;

    &#125;)
</script>

v-bind

设置元素属性（例：src，titlt，class）

完整写法：v-bind：属性名=”内容”，简写可以省略v-blind，只保留 ：属性名

如果动态的增删class，建议使用对象的方式。

<div id="app">
    <img v-bind:src="imgSrc" v-bind:title="imgTitle+'!!!'"> 

    <img :src="imgSrc" :title="imgTitle+'!!!'" 
    :class="isActive?'active':''" @click="toggleActive"> <!--简写可以省略v-bind，只留下：-->

    <img :src="imgSrc" :title="imgTitle+'!!!'" 
    :class="&#123;active:isActive&#125;" @click="toggleActive">
    <!—对象方式：&#123;active:isActive&#125;表示是否等于active取决于isActive是否等于true-->

</div>



<script>
    let app=new Vue(&#123;
      el:"#app",

      data:&#123;
        imgSrc:"Images/325530.jpg",
        imgTitle:"图片",
        isActive:false
      &#125;,

      methods:&#123;
        toggleActive:function()&#123;   //点击切换isActive的值
          this.isActive=!this.isActive;
        &#125;,        
      &#125;

    &#125;)
</script>

v-for

根据数据生成列表结构,数组经常和v-for结合使用。语法是（item,index） in arr。Item和index可以结合其他指令一起使用。数组长度的更新会同步到页面上，是响应式的。

<body>
    <div id="app">
        <ul>
            <input type="button" value="添加数据" @click="add">
            <input type="button" value="移除数据" @click="remove">

            <li v-for="(item,index) in arr" :title="item">  
            <!--(item,index)对应元素值和索引-->
                &#123;&#123;index+1+"."&#125;&#125; &#123;&#123;item&#125;&#125;
            </li>
            <li v-for="(item,index) in heros" :title="item.name">  
                <!--objArr中存的是对象-->
                名字：&#123;&#123;item.name&#125;&#125;
            </li>
        </ul>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <script>
       let app= new Vue(&#123;

           el:"#app",
           data:&#123;
               arr:["德莱文","杰斯","诺手","艾希","蛮王"],

               heros:[
                   &#123;name:"jack"&#125;,
                   &#123;name:"rose"&#125;
               ]
           &#125;,

           methods:&#123;
               add:function()&#123;
                   this.heros.push(&#123;name:"jack"&#125;);
               &#125;,
               remove:function()&#123;
                   this.heros.shift();
               &#125;
           &#125;

       &#125;)
    </script>

v-on补充

传递自定义参数，事件修饰符。事件绑定的方法写成函数调用的形式，可以传入自定义参数。

定义方法时需要定义形参来接收传入的实参。事件的后面可以跟上.修饰符可以对事件进行限制。

<div id="app">
        <!--v-on可以传递多个参数，在对应的函数方法中也要设置相应的形参-->
       <input type="button" value="点击" @click="doIt(666,'老铁')">

       <!--v-on也可以使用事件修饰符-->
       <!--keyup是修饰符，指键盘按钮弹起时触发事件；设置具体按钮可以写成 keyup.特定按钮-->
       <input type="text" @keyup.enter="sayHi">
    </div>

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <script>
        let app= new Vue(&#123;
            el:"#app",
            methods: &#123;
                doIt:function(p1,p2)&#123;   
                    console.log("做IT");
                    console.log(p1);
                    console.log(p2);
                &#125;,

                sayHi:function()&#123;
                    alert("吃了没");
                &#125;
            &#125;
        &#125;)

    </script>

更多修饰符可以参考官方文档：

https://cn.vuejs.org/v2/api/#v-on

v-model

获取和设置表单元素的值（双向数据绑定），绑定的数据会和表单元素值相关联。

<div id="app">
        <!--v-model获取表单元素的内容-->
        <!--在表单中修改值时，对应data中的值也会被同步修改-->
        <input type="text" v-model="message" @keyup.enter="getM">

        <!--修改data中的值时，表单中的值也会被修改-->
        <input type="button" value="修改message" @click="setM">
        <h2>&#123;&#123;message&#125;&#125;</h2>
</div>

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <script>
        let app= new Vue(&#123;
            el:"#app",

            data:&#123;
                message:"Vue"
            &#125;,

            methods: &#123;
                getM:function()&#123;
                    alert(this.message);
                &#125;,

                setM:function()&#123;
                    this.message ="酷丁鱼";
                &#125;
            &#125;
        &#125;)

    </script>
