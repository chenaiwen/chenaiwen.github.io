Array基础

创建一个数组

//字面量方法：
let arr=[1,2,3];

//第二种：
let arr1=Array();
let arr2=Array(3); //固定数组大小
let arr3=Array(1,2,3)//直接赋值

//new方法
let arr4=new Array();
let arr5=new Array(3);
let arr6=new Array(1,2,3);

//Array()和new Array()没有什么大的区别,当初始化一个空数组时候，两者性能几乎没有差异。
//从语言特性上讲，JS是一个基于原型继承的语言。无论是new的引入，以及在ES6中引入的Class语法糖，都
//是为了将JS进一步向对象化的方向进行过度。虽然使用new，会增加多一层的对象包裹，而使得内存冗余,但使用了new后更加符合了对象化继承的概念。

构造函数上的方法

Array.of()

创建一个新数组的实例

语法：

Array.of(element0[, element1[, ...[, elementN]]])

用法：

Array.of(3);         //[3]
Array.of(1,2,3);     //[1,2,3]

与Array的区别：

Array.of(3)是创建一个具有单个元素7的数组，而Array(7)是创建一个长度为7的空(empty)数组。

Array.isArray()

用于确定传递的值是否是一个Array。

Array.isArray([1,2,3]);    //true
Array.isArray(&#123;foo: 123&#125;); //false
Array.isArray(undefined);  //false

Array.from()

从一个类似数组或可迭代对象创建一个新的、浅拷贝的数组实例。

语法：

Array.from(arrayLike, mapFunction, thisArg])

参数：

  1. arrayLike：必选，要转换成数组的对象。可以传入类数组（argument）和可迭代对象（set， map）。
  2. mapFunctionn：可选，数组中每个元素都要调用的对象。相当于Array.from(arrayLike).map(mapFunction, thisArg)。
  3. thisArg：可选，映射函数(mapFunction)中的 this 对象。

用法：

1.将字符串转换成数组

let str='hello!';
let arr=Array.from(str);
console.log(arr)  //['h','e','l','l','o']

2.将set结构的数据转换成数组

let arr=Array.from(new Set([1,2,3,4,1,2,3]));
console.log(arr);  //[1,2,3,4]; 同时会去重。

3.map()

let map=new Map([[1,2],[2,4],[3,4]]);
Array.from(map);  //[[1,2],[2,4],[3,4]]

4.类数组

function fn()&#123;
    return Array.from(arguments);
&#125;
fn(1,2,3);   //[1,2,3]


let array = &#123;
    0: 'name', 
    1: 'age',
    2: 'sex',
    3: ['user1','user2','user3'],
    'length': 4     //这里的length属性值决定返回数组的大小，若没有length属性，则返回空数组。
&#125;//这里对象的属性名必须是数字，否则返回的数组为undefined。
let arr = Array.from(array )
console.log(arr)        // ['name','age','sex',['user1','user2','user3']]

5.对数组进行操作

let arr=[1,2,3];
let set=new Set(arr);
Array.from(set, item=>item+1);// 类map方法，每个元素进行处理，将处理后的值放入返回的数组。
                              //[2,3,4]

6.thisArg

let arr=Array.from([1,2,3], function(x)&#123;
    return x+ this.a
&#125;,&#123;a:3&#125;);
console.log(arr);  //[4,5,6]
                   //function里的this就指向了&#123;a:3&#125;;

const obj=&#123;
    handle: x=>x*4
&#125;
console.log(Array.from([1,2,3], function(x)&#123;
    return this.handle(x)
&#125;, obj));          //[4,8,12]
                   //function里的this指向obj,调用obj里面的handle(x)方法

常见方法

遍历方法

forEach()

语法：

array.forEach(func(currentValue, index, arr), thisArg)

参数：

func(): 数组中每个元素需要调用的函数。

​ currentValue: 必需，数组中正在处理的当前元素

​ index: 可选，数组中正在处理的当前元素的索引

​ arr: 可选，forEach() 方法正在操作的数组

thisArg: 可选参数,当执行回调函数,用作this值

let arr = [1,2,3];
let sum=0;
arr.forEach(function(value, index, array)&#123;
    array[index]==value;
    sum+=value;
&#125;);

console.log(sum);   //10

every()

测试一个数组内所有元素是否都能通过某个指定函数的测试。它返回一个布尔值。

语法：

array.every(func(currentValue, index, arr), thisArg)

参数：

func: 数组中每个元素执行的函数，该函数接收一至三个参数。

​ currentValue：必需，数组中正在处理的当前元素

​ index：可选，数组中正在处理的当前元素的索引

​ arr：可选，every() 方法正在操作的数组

thisArg：可选参数,当执行回调函数func, 用作this值

function isBigEnough(element,index,array)&#123;
    return element >=10;
&#125;

[12,5,8,30].every(isBigEnough);   //false
[12,15,32,48].every(isBigEnough); //true

注意：

  1. 空数组的情况下，只要第一个参数是回调函数，一切情况返回为true。
  2. 要每次返回值都为true,最后返回true,否则为false。
  3. every不会改变原数组。

some

测试数组中是不是至少有1个元素通过了被提供的函数测试。它返回的是一个Boolean类型的值。

语法：

array.some(func(currentValue, index, arr), thisArg)

参数：

func: 数组中每个元素执行的函数，该函数接收一至三个参数。

​ currentValue：必需，数组中正在处理的当前元素。

​ index：可选，数组中正在处理的当前元素的索引。

​ arr：可选，some() 方法正在操作的数组。

thisArg：可选参数,当执行回调函数func, 用作this值。

function isBiggerThan10(element, index, array)&#123;
    return element>10;
&#125;
[2, 5, 8, 1, 4].some(isBiggerThan10);  // false
[12, 5, 8, 1, 4].some(isBiggerThan10); // true

/*--------------------------------------------------------*/

let fruit=['apple', 'banana', 'mango', 'guava'];

function checkAvailability(arr, val)&#123;
    return arr.some(function(arrVal)&#123;
        return val===arrVal;
    &#125;);
&#125;
checkAvailability(fruits, 'kela');    // false
checkAvailability(fruits, 'banana');  // true

注意：

  1. some不会改变原数组.
  2. 如果用一个空数组进行测试，在任何情况下它返回的都是false。
  3. 如果你回调函数没有返回值，每次都是undefined，最后调用some结果返回也是false.

filter

定义：

创建一个新数组, 其包含通过所提供函数实现的测试的所有元素。

语法：

let newArray = array.filter(func(currentValue, index, arr), thisArg)

参数：

func: 数组中每个元素执行的函数，该函数接收一至三个参数。

​ currentValue：必需，数组中正在处理的当前元素。

​ index：可选，数组中正在处理的当前元素的索引。

​ arr：可选，filter() 方法正在操作的数组。

thisArg：可选参数,当执行回调函数func, 用作this值。

function isBigEnough(element)&#123;
    return element >=10;
&#125;
let filtered=[12,3,15,9,11].filter(isBigEnough);  //[12,15,11]

/*-------------------------------------------------------------*/

let fruits = ['apple', 'banana', 'grapes', 'mango', 'orange'];
//字符串匹配
function filterItems(query)&#123;
    return fruits.filter(function(el)&#123;
        return el.toLowerCase().indexOf(query.toLowCase()) > -1
    &#125;)
&#125;

console.log(filterItems('ap')); // ['apple', 'grapes']
console.log(filterItems('an')); // ['banana', 'mango', 'orange']

注意：

  1. 自定义回调函数要有Boolean返回值，不写默认返回undefined，则转Boolean为false。
  2. 不会修改原始数组，但是会返回一个新数组，包含通过所提供函数实现的测试所以元素。
  3. 没有任何元素通过的话，返回空数组。

map

创建一个新数组，其结果是该数组中的每个元素是调用一次提供的回调函数后的返回值。

语法：

let newArray = array.map(function(currentValue, index, arr), thisArg);

参数：

func: 数组中每个元素执行的函数，该函数接收一至三个参数。

​ currentValue：必需，数组中正在处理的当前元素。

​ index：可选，数组中正在处理的当前元素的索引。

​ arr：可选，map() 方法正在操作的数组。

thisArg：可选参数,当执行回调函数func, 用作this值。

let numbers = [1,4,9];
let roots=numbers.map(Math.sqrt);  //[1,2,3]

/*----------------------------------------------*/

let numbers = [1, 4, 9];
let doubles=numbers.map(function(num)&#123;
    return num*2;
&#125;);       //[2, 8, 18]

注意：

  1. map不修改调用它的原数组本身。
  2. 回调函数不返回值时，最后新数组的每个值都为undefined。
  3. 因为map生成一个新数组，当你不打算使用返回的新数组，请用forEach或者for-of替代。

reduce

对数组中的每个元素执行一个用户提供的「reducer」函数(升序执行)，将其结果汇总为单个返回值。

语法：

let result = array.reduce(func(accumulator, currentValue, currentIndex,                        array), initialValue)

参数：

func: 为数组中每个元素执行的函数，该函数接收一至4个参数

​ accumulator: 累计器

​ currretValue：当前值

​ currentIndex：当前索引

 array：数组

initialValue：作为第一次调用 func函数时的第一个参数的值。 如果没有提供初始值，则将使用数组中 的第一个元素。 在没有初始值的空数组上调用 reduce 将报错。

let arr=[1,2,3];
let a=arr.reduce((t,v)=>t+v);   //6
//等同于
let b=arr.reduce((t,v)=>t+v,0);

注意：

  1. 回调函数第一次执行时，accumulator 和currentValue的取值有两种情况：如果调用reduce()时提供了initialValue，accumulator取值为initialValue，currentValue取数组中的第一个值；如果没有提供 initialValue，那么accumulator取数组中的第一个值，currentValue取数组中的第二个值。
  2. 如果数组为空且没有提供initialValue，会抛出TypeError 。

reduceRight

接受一个函数作为累加器（accumulator）和数组的每个值（从右到左）为单个值，将其结果汇总为单个返回值。

语法：

let result=array.reduceRight(func(accumulator, currentValue, currentIndex),                 initialValue)

参数：

func: 为数组中每个元素执行的函数，该函数接收一至4个参数

​ accumulator: 累计器

​ currretValue：当前值

​ currentIndex：当前索引

 array：数组

initialValue：作为第一次调用 func函数时的第一个参数的值。 如果没有提供初始值，则将使用数组中 的第一个元素。 在没有初始值的空数组上调用 reduce 将报错。

let a = ['1', '2', '3', '4', '5'];
let left=a.reduce(function(prev, cur)&#123;return prev+cur;&#125;);
let right=a.reduceRight(function(prev, cur)&#123;return prev+cur&#125;);

console.log(left);  // "12345"
console.log(right); // "54321"

find

返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined。

语法：

let elem=array.find(func(element, index, arr), thisArg);

参数：

func: 为数组中每个元素执行的函数，该函数接收一至三个参数
elemnet: 数组中正在处理的当前元素
index: (可选) 数组中正在处理的当前元素的索引
arr: (可选) find方法正在操作的数组
thisArg: 可选参数,当执行回调函数callback,用作this值

function isPrime(element, index, array)&#123;
    let start=2;
    while(start<=Math.sqrt(element))&#123;
        if(element % start++<1)&#123;
            return false;
        &#125;
    &#125;
    return element>1;
&#125;

console.log([4, 6, 8, 12].find(isPrime)); // undefined, not found
console.log([4, 5, 8, 12].find(isPrime)); // 5

参数：

func: 为数组中每个元素执行的函数，该函数接收一至三个参数
elemnet: 数组中正在处理的当前元素
index: (可选) 数组中正在处理的当前元素的索引
arr: (可选) find方法正在操作的数组

keys & values & entries

keys()方法返回一个包含数组中每个索引键的Array iterator对象。

values()方法返回一个新的 Array Iterator对象，该对象包含数组每个索引的值。

entries()方法返回一个新的Array Iterator对象，该对象包含数组中每个索引的键/值对。

语法：

arr.keys();
let arr=['a', 'b', 'c'];
let iterator1=arr.keys();
let iterator2 = arr.values();
let iterator3 = arr.entries();
console.log(iterator1);

//使用for...of输出一下iterator的值
for (let e of iterator3) &#123;
    console.log(e);         
&#125;
// [0, "a"] 
// [1, "b"] 
// [2, "c"]
